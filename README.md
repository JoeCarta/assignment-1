# Personal Portfolio — Joseph Cartafalsa

A small, hand-written personal portfolio site built for CSCI 39548 (Practical Web
Development) Assignment 1. No frameworks, no build tools — just semantic HTML and CSS.

## About

A single-page portfolio for Joseph Cartafalsa, a computer science student at Hunter
College. It includes an About section, a Projects section, and a Contact form, with a
responsive layout that adapts between phone and desktop widths.

## Built with

- Semantic HTML5 (`header`, `nav`, `main`, `section`, `footer`)
- Plain CSS with a flexbox/grid layout and a `@media` query for responsiveness
- An inline SVG avatar (no binary assets required)

## Running it

Open `index.html` in any web browser — there is no build step.
